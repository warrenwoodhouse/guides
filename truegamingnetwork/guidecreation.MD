Here's an updated list of forum formatting, Replace { } with [ ].

# Text Formatting
* Bold - {b}text{/b}

* Italic - {i}text{/i}

* Underline - {u}text{/u}

* Strikethrough - {s}text{/s}

* Superscript - {p}text{/p}

# Link Formatting
* Hyperlinked text - {url=url you wish to link here}text for link here{/url}

* To disregard the site's automatic formatting of achievements, example Homerun! in Fallout 4 - {noformat}link{/noformat}

# Quote Formatting
* Quoting - {quote}Text{/quote} creates a quote box with no gamer

* Quoting members- {quote gamer=gamertag here}Text{/quote} creates a quote box with a gamer prefix

# Spoiler Formatting
* Spoiler - {spoiler}Spoiler here{/spoiler}
Spoiler - click to reveal

# Chart Formatting
A final text formatting is the site's chart function which has many uses for things like blogs.

* {chart }{/chart} - This will create a thin line that spans the post box

* {chart perc=100}Text{/chart} - This creates a chart with text, which many of our members use for headers in their blog, you can change the 100 to choose how much the chart is filled by.

* {chart perc=100 end=123}Text{/chart} - This creates a chart similar to the one previous but instead adds a figure at the end, which is used for the charts original function as a chart.

Charts can also be hyperlinked by wrapping them in the url encoding detailed earlier

# Image Formatting
A final one for use in blogs, solutions and reviews is Image formatting.

* {img=imageurl label=true}Label!{/img} For an image with a caption

* {img=imgurl style=left}{/img} to make the image align left or right

You can combine the caption and alignments.

# Video Formatting
* {url=}YouTube video url goes here{/url} to add a YouTube video, you can also add {url= style=left} or {url= style=right} to format the video to the left or right, however if left untouched, the video will be centered.
